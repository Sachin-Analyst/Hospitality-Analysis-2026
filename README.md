## Project Description
This repository contains a specialized hospitality analytics suite that provides a holistic view of hotel property performance. The dashboards and models evaluate critical KPIs including revenue , Realisation , Daily Sellable Room Nights (DSRN), Revenue Per Available Room (RevPAR), Occupancy %, and Average Daily Rate (ADR) to improve room utilization, assess platform profitability, and support strategic growth for the luxury hotel chain.

## Folder Structure
- *Colour_Idea's* - The colors and themes I chose to make the dashboard look good and added the inspiration ideas where it generated
- *Raw_Data* - The actual  files and data I used to build the project.
- *Report* - The finished dashboard so you can see the results in formate PDF
- *Resources* Extra stuff like icons used in the Hospitality Analysis
- *Solution* The final results in the function added as the format by Screen Recorded
- *Deleted Scenes* A collection of initial design iterations, early project concepts, and visualizations that were explored but ultimately removed to refine the final dashboard

## Installation

To explore or modify this Excel project:

1. *Clone the repository:*   
   ```bash

    git clone https://github.com/Sachin-Analyst/Hospitality-Analysis-2026
`
   - Open terminal and run the command


2. *Download and Open MS Excel*
   - This analysis can be done using any versions of excel
   - This analysis can be done using any versions of Power BI

3. *Explore Resources*
     - Open the Resources folder to review the data dictionary and the logic behind critical hospitality metrics like RevPAR, ADR, and Realization %


4. *Build the Report*
    - Navigate to the Raw Data folder and download the Excel files and Connect to Power BI
